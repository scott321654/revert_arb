# 台灣50指數成分股調整 — 尾盤均值回歸套利策略

基於台灣50指數（0050/006208）季度調整的「尾盤甩尾效應」量化套利策略。

## 策略原理

台灣50指數每年3/6/9/12月進行季度審核，於當月第三個週五收盤後生效。被動型ETF（0050、006208）為降低追蹤誤差，會在生效日**13:25-13:30**收盤集合競價時段集中換股，造成：

- **被剔除的個股** → 單向賣壓 → 尾盤超跌 → **隔日開盤均值回歸**
- **被納入的個股** → 單向買盤 → 尾盤超漲 → 隔日回落

## 重要發現（回測14次）

| 項目 | 數據 |
|------|------|
| 做多超跌股勝率 | **88.89%** |
| 做空超漲股勝率 | **0.00%** |
| 平均單次淨報酬（做多） | **+1.72%** |
| 雙邊交易成本 | ~0.357% |

> ⚠️ **只做多，不做空。** 做空超漲股全部虧損，直接放棄。

## 使用方式

### 0. 事前準備：輸入本次調整名單（可略過，非必須）

審核日公告後，先記錄本次刪/納入名單，`check` 時就不用再選事件類型：

```bash
python3 run.py adjust

=== 輸入本次調整名單 ===
季度 (如 2026Q3): 2026Q3
被剔除的股票代號 (逗號分隔): 1301,2002
新納入的股票代號 (逗號分隔): 3037,3665,8046
權重調降的代號 (選填): 
```

### 1. 查詢下次調整時程

```bash
python3 run.py schedule
```

輸出範例：
```
今日日期: 2026-07-02
交易成本: 0.357% (雙邊)
策略方向: 只做多超跌股

     2026Q3  審核日: 2026-09-04  生效日: 2026-09-18 ← 下一次
     2026Q4  審核日: 2026-12-04  生效日: 2026-12-18
```

### 2. 生效日尾盤檢查訊號（自動模式）

什麼都不用算，程式直接從證交所拉報價 + 算波動率：

```bash
python3 run.py monitor 5871
```

```
📡 正在從證交所查詢 5871 即時報價...
✅ 5871 中租-KY
   現價: 115.5
   昨收: 114.5
   今日: 開 114.0 高 116.5 低 113.5
📊 今日漲跌: +0.87%
```

- 自動抓**即時股價**、**昨收**、**今日高低**
- 自動計算**近5日日內波動率**
- 自動從 `adjust` 名單辨識事件類型（剔除/納入）
- 若今天收跌，自動帶入偏離度；若收漲也可手動修正

### 3. 生效日尾盤檢查訊號（手動模式）

如果沒網路或想自己算數字：

```bash
python3 run.py check 5871 -3.9 1.2
```

- 第一個參數 = 股票代號（名稱自動查詢）
- 第二個參數 = 尾盤5分鐘跌幅
- 第三個參數 = 近5日日內波動率%

### 4. 隔日開盤平倉

```bash
python3 run.py exit
```

### 5. 查詢績效統計

```bash
python3 run.py summary
```

### 6. 查詢目前持倉

```bash
python3 run.py positions
```

## 策略篩選條件（程式自動判斷）

| 條件 | 說明 |
|------|------|
| Z-score ≥ 2.5σ | 尾盤跌幅需超過個股日內波動率的2.5倍標準差 |
| 方向 = Long | 僅接受超跌買進（負偏差） |
| 非處置股/注意股 | 處置股分盤撮合破壞套利流動性 |
| 融券比率 < 25% | 避免空單過度擁擠（如台塑2026/6案例） |

## 交易成本計算

## 風險提示

1. **隔夜系統性風險**：美股若隔夜大跌，台股隔日跳空會淹沒均值回歸收益（建議搭配台指期貨避險）
2. **擁擠交易反噬**：當所有人都預期尾盤會跌而搶進時，可能發生反向甩尾（如台塑案例）
3. **流動性風險**：張數過大可能造成滑價，小額交易較安全
4. 本程式僅提供**訊號產生與交易記錄**，實際下單請自行操作
