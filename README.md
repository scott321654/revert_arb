# 台灣50指數成分股調整 — 尾盤均值回歸套利策略

基於台灣50指數（0050/006208）季度調整的「尾盤甩尾效應」量化套利策略。

## 策略原理

台灣50指數每年3/6/9/12月進行季度審核，於當月第三個週五收盤後生效。被動型ETF（0050、006208）為降低追蹤誤差，會在生效日**13:25-13:30**收盤集合競價時段集中換股，造成：

- **被剔除的個股** → 單向賣壓 → 尾盤超跌 → **隔日開盤均值回歸**
- **被納入的個股** → 單向買盤 → 尾盤超漲 → 隔日回落

## 重要發現（回測14次）

| 項目 | 數據 |
|------|------|
| 做多超跌股勝率 | **88.89%** |
| 做空超漲股勝率 | **0.00%** |
| 平均單次淨報酬（做多） | **+1.72%** |
| 雙邊交易成本 | ~0.357% |

> ⚠️ **只做多，不做空。** 做空超漲股全部虧損，直接放棄。

## 零股投資人實戰執行方式

若資金不足以買整張（1000 股），可用以下方式參與此策略：

### 方案一：盤後零股交易（當天 13:40 ~ 14:30）

下單軟體選擇「**盤後零股**」（非盤中零股，也非盤後定價），手動輸入 **13:30 收盤價**，系統於 **14:30** 統一撮合：

```
✅ 優點：有機會以尾盤超跌價成交
⚠️ 缺點：盤後零股流動性低，不保證成交
```

### 方案二：隔日開盤買「盤中零股」⭐ 推薦

當天 13:30 先看戲確認訊號，隔日 **09:00~09:15 開盤期間**用「**盤中零股**」買進：

```
✅ 優點：
   - 每 5 秒撮合一次，流動性極佳，保證買得到
   - 隔日開盤常有「餘震」開低，有機會買到比前日收盤更便宜的價格
⚠️ 缺點：若前日美股大漲導致台股隔日開高，可能買在較高位置
```

> 兩種方案皆可搭配本程式的訊號使用，差別只在執行時間點不同。

## 使用方式

### 0. 事前準備：輸入本次調整名單（可略過，非必須）

審核日公告後，先記錄本次刪/納入名單，`check` 時就不用再選事件類型：

```bash
python3 run.py adjust

=== 輸入本次調整名單 ===

被剔除的股票代號 (逗號分隔): 1301,2002
新納入的股票代號 (逗號分隔): 3037,3665,8046
權重調降的代號 (選填): 
```

### 1. 查詢下次調整時程

```bash
python3 run.py schedule
```

輸出範例：
```

交易成本: 0.357% (雙邊)
策略方向: 只做多超跌股

     2026Q4  審核日: 2026-12-04  生效日: 2026-12-18
```

### 2. 生效日尾盤檢查訊號（批次自動模式）

先 `adjust` 輸入批次名單，生效日 13:00 前跑一次即可：

python3 run.py monitor all       # 批次掃描所有剔除/調降股
```

程式會自動：

```

## 策略篩選條件（程式自動判斷）

| 條件 | 說明 |
|------|------|
| Z-score ≥ 2.5σ | 尾盤跌幅需超過個股日內波動率的2.5倍標準差 |
| 方向 = Long | 僅接受超跌買進（負偏差） |
| 非處置股/注意股 | 處置股分盤撮合破壞套利流動性 |
| 融券比率 < 25% | 避免空單過度擁擠（如台塑2026/6案例） |

## 風險提示

1. **隔夜系統性風險**：美股若隔夜大跌，台股隔日跳空會淹沒均值回歸收益（建議搭配台指期貨避險）
2. **擁擠交易反噬**：當所有人都預期尾盤會跌而搶進時，可能發生反向甩尾（如台塑案例）
3. **流動性風險**：張數過大可能造成滑價，小額交易較安全
4. 本程式僅提供**訊號產生與交易記錄**，實際下單請自行操作
